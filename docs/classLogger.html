Returns different combinations of the date and time.

Return the time so it can be used by parts of the logger. The method can be called such that the user has control over which parts of the date get returned. For example, they can ask for the format yyyy-dd, mm-dd_hh-mm-ss, dd_hh-mm-ss, or any other combination. The time scale gets smaller the farther right the line goes. Time (hh-mm-ss) uses the delimiter '-' instead of ":" because Windows does not allow ":" in file names.

Parameters
yearReturn the year?
monthReturn the month?
dayReturn the day?

Returns
The current date and time of the system.
