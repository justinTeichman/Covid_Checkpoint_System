CredentialsVerifier::CredentialsVerifier ( )

This class verifies the login credentials entered by the user.

This class is used to verify the login credentials that the user entered into the login GUI. It reads the valid usernames and passwords from the "CredentialsDatabase.txt" and compares them to the users entered credentials, and returns if they're valid or not.
