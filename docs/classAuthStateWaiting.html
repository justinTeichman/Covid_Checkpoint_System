Detailed Description

Header for visual authentication state implementations in AuthUI.

These concrete classes implement an interface between AuthUI and various visual states associated with feedback provided to the user. Each AuthState overrides methods that define what changes should be made to the user interface how long the state should remain on screen, plus any logging associated with the event that brought about the state.
